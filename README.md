# ①課題名
ポケモンバトル（続）

## ②課題内容（どんな作品か）
-前回までは対トレーナーとの戦闘をイメージしたポケモンバトルを作りましたが、今回は野生のポケモンと戦っているようなポケモンバトルを作りました。

## ③工夫した点・こだわった点
- 工夫した点：
 最初にたたかう・ポケモン・どうぐ・にげるのボタンを用意しておいて、もっとバトルの自由度が高くなる様にしました。
　プレイヤーが捕獲したポケモンのデータをローカルストレージに保存しておいて、次回から再読み込みしたときにそのポケモンが手持ちに表示される様にしました（最大６匹まで）

作成時間があまり取れず、こだわる様なポイントを作り上げることができませんでした。

## ④難しかった点・次回トライしたいこと（又は機能）
- 全体的なクオリティがまだまだと感じました。
- 手持ちがいっぱいだった時に、モンスターボックスの様なところに保存させておきたいと思いました
- 道具に個数制限がある様に設定したいです
