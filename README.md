# ①課題名
ポケモンバトル（続）

## ②課題内容（どんな作品か）
-前回までは対トレーナーとの戦闘をイメージしたポケモンバトルを作りましたが、今回は野生のポケモンと戦っているようなポケモンバトルを作りました。マップ上をユーザーが移動して、ある一定のエリアのところでランダムにバトルが発生する様に設定しました。

## ③工夫した点・こだわった点
- 工夫した点：
 最初にたたかう・ポケモン・どうぐ・にげるのボタンを用意しておいて、もっとバトルの自由度が高くなる様にしました。
-プレイヤーが捕獲したポケモンのデータをローカルストレージに保存しておいて、次回から再読み込みしたときにそのポケモンが手持ちに表示される様にしました（最大６匹まで）
-トレーナーがいる場所に応じて出現するポケモンが変わる様に設定しました



## ④難しかった点・次回トライしたいこと（又は機能）
- 全体的なクオリティがまだまだと感じました。
- 道具をデータとして保存しておいて、自由に取り出しができるように設定したいと思いました
- 道具に個数制限がある様に設定したいです
-ポケモンセンターのhtmlを作って、ユーザーが入れる様にしたいと思いました。
-技のダメージを計算するときに、ある程度のランダム要素を追加したいと思いました
-回復道具を使用する際に、どのポケモンに使用するか選べる様にしたいと思いました
-ポケモンを交代するときに、ユーザー側はランダムにするのではなく、選べる様な設定にしたいと思いました
-NPCと対戦できる要素を追加したいと思いました
